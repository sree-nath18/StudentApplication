Student ManageMent System(ASP.NET MVC)
his is a web-based student management system I developed using ASP.NET MVC and Entity Framework.
It mainly focuses on managing student details and login activities. Students can register, log in, view their profile. Admin can log in separately to view, edit, and manage all student records from one place.

     Technologies Used
Frontend: HTML, CSS, Bootstrap
Backend: ASP.NET MVC 5, C#
Database: SQL Server
ORM: Entity Framework (Code First)
Authentication: Session-based login
Version Control: Git and GitHub

Project Overview
The application follows the MVC architecture.
It has separate controllers for Student and Admin operations, models for handling data, and Razor views for displaying pages.
The main goal of this project is to understand how MVC works with Entity Framework and how to perform CRUD operations in a structured way.
